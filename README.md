# 🎓 Student Management System (Ruby)

A command-line based Student Management System developed in Ruby that performs CRUD operations and stores student data using file handling.

## ✨ Features

* ➕ Add Student
* 📋 View Students
* 🔍 Search Student by ID
* ✏️ Update Student Details
* ❌ Delete Student
* 💾 Store Data in Text Files

## 🛠️ Technologies Used

* Ruby
* Object-Oriented Programming (OOP)
* File Handling
* CRUD Operations

## 🚀 How to Run

1. Clone the repository

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder

```bash
cd student-management-system
```

3. Run the application

```bash
ruby main.rb
```
